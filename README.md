## 🛡️ SIEM – Security Information & Event Management (Splunk Labs)

This repository contains a structured collection of hands-on **Splunk SIEM labs** designed for SOC (Security Operations Center) training and real-world security monitoring practice.

The labs simulate enterprise-level security operations including SIEM deployment, log ingestion, dashboard creation, correlation rule development, alerting, and proactive threat hunting using SPL (Search Processing Language).

It is built to reflect practical SOC workflows and helps develop skills required for junior SOC analysts, SIEM engineers, and security operations roles.

---

## 🎯 What This Repository Covers

This project focuses on core SIEM and SOC engineering capabilities:

* Deployment and configuration of Splunk Enterprise SIEM
* Ingestion and normalization of security logs from Windows and Linux systems
* Building SOC dashboards for real-time monitoring and visibility
* Creating correlation searches and security alerts
* Writing SPL queries for detection engineering
* Performing threat hunting using hypothesis-driven analysis
* Mapping detections to attacker behaviors and security use cases
