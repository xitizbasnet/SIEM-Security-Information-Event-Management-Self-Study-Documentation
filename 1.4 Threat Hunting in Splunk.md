## 🎯 Objective

Perform **proactive, hypothesis-driven threat hunting** using Splunk SPL queries to identify advanced adversary behaviors, including lateral movement, persistence, and PowerShell-based attacks.

---

# 📌 STEP 1: Hunt for Pass-the-Hash (PTH) Attacks

---

## 🧠 Hypothesis

An attacker is using **NTLM hash-based authentication** to move laterally across systems without using plaintext credentials.

---

## 🔍 SPL Query

```spl id="pth1"
index=security_logs EventCode=4624 Logon_Type=3 Authentication_Package=NTLM
| stats count by user, src_ip, ComputerName
| sort -count
```

---

## 🎯 Investigation Focus

Look for:

* Service accounts authenticating from **unusual endpoints**
* Multiple hosts accessed by same credentials
* NTLM authentication outside normal baselines

---

# 📌 STEP 2: Hunt for PowerShell Abuse

---

## 🧠 Objective

Detect malicious or suspicious PowerShell execution used for:

* Payload download
* Command execution
* Obfuscation

---

## 🔍 Process Execution Monitoring

```spl id="ps1"
index=security_logs sourcetype=WinEventLog:Security EventCode=4688
process=*powershell*
| table _time, user, CommandLine, ComputerName
```

---

## 🚨 Suspicious Indicators

Flag commands containing:

* `-EncodedCommand`
* `-WindowStyle Hidden`
* `IEX` (Invoke-Expression)
* `DownloadString`
* `Bypass`

---

## 🧪 Script Block Logging (Deeper Inspection)

```spl id="ps2"
index=security_logs EventCode=4104
| search ScriptBlockText=*DownloadString* OR ScriptBlockText=*Invoke-Expression*
```

---

# 📌 STEP 3: Hunt for Persistence Mechanisms

---

## 🧠 Objective

Identify attacker persistence techniques used to maintain access after reboot or credential changes.

---

# 🗓️ Scheduled Tasks Persistence

```spl id="per1"
index=security_logs EventCode=4698
| table _time, user, TaskName, TaskContent
```

---

# 🧬 Registry Run Keys Persistence

```spl id="per2"
index=security_logs EventCode=13 TargetObject=*Run*
| table _time, user, TargetObject, Details
```

---

# ⚙️ New Service Creation

```spl id="per3"
index=security_logs EventCode=7045
| table _time, ServiceName, ImagePath, StartType
```

---

## 🚨 Key Persistence Indicators

* Unknown scheduled tasks
* Execution from unusual directories (`Temp`, `AppData`)
* New services with suspicious binary paths
